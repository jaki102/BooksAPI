# BooksAPI
>Application to show books from database with the possibility of sorting them.
## Technologies
* JavaScript: ES5/ES6
* Vue: 2.6.10
* Express.js: 4.17.1
* MySQL: 2.17.1
## Setup
### Backend
from the level of the server folder
```
npm install
nodemon or node index.js
```
### Frontend
from the level of the client folder
```
npm install
npm run serve
```
